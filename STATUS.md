# Project Status

FieldProof is in incubation.

## Maturity

- Prototype: yes
- Production app: no
- Production contracts: no
- Security audit: no
- Pilot deployment: no
- Real audit data: no

## What To Expect

This repository is useful for:

- Reviewing the product direction.
- Critiquing the audit lifecycle.
- Contributing architecture notes.
- Prototyping screens and schemas.
- Planning a pilot.

This repository is not yet useful for:

- Running real farmland audits.
- Making compliance claims.
- Paying auditors.
- Storing sensitive field evidence.
- Proving environmental truth.

## Near-Term Goal

Turn the clickable prototype into a buildable MVP specification, then build the smallest working vertical slice:

1. Auditor captures a sample audit packet.
2. Evidence is hashed locally.
3. Evidence metadata is stored in a portable JSON schema.
4. A public dashboard can inspect the sample record.
