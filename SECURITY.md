# Security Policy

FieldProof is not production-ready and has not been security reviewed.

Do not use this repository yet for:

- Real private farm data.
- Real payment flows.
- Real identity documents.
- Real compliance claims.
- Sensitive field-worker location data.

## Reporting Security Issues

Until the project has a formal maintainer group, please open a GitHub issue with a high-level description and avoid posting exploit details publicly.

Future versions should add:

- Private security contact.
- Threat model.
- Responsible disclosure process.
- Dependency scanning.
- Smart contract review process.

## Known Risk Areas

- GPS spoofing.
- AI-generated or edited media.
- Device compromise.
- Auditor coercion or retaliation.
- False environmental claims.
- Validator conflicts of interest.
- Public release of sensitive location data.
- Smart contract escrow bugs.
